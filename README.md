# Femboy Spectrum Assessment (FSA)
basically a femboy rating website :3

#### REQUIREMENTS:
- PHP `https://marketplace.visualstudio.com/items?itemName=DEVSENSE.phptools-vscode`

#### HOW TO RUN:

1. Open xampp and run Apache.
2. Clone this repository via vscode
3. Store the repository in your xampp folder (i.e. `C:\xampp\htdocs\`)
4. Open your browser and type `localhost`
5. navigate to the repo
6. Test
